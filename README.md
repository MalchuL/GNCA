## Installation
All experiments was running **Ubuntu OS** NVDIA 2080 TI GPU.

Requirements:

    * Python 3.6+
    * CUDA 9.1+
    * 10GB GPU Memory 
   
    
Steps to reproduce:

    1. Run `pip install -r requirements.txt`.   
    2. Run `python main.py`
